About NEXUS
Microsoft Innovation Studio · Creative Apps Track
Creative Apps GitHub Copilot Work IQ Chaos Math
A Civilization Collapse Simulator built for the Microsoft Agents League Hackathon under the Creative Apps track. NEXUS transforms real-world scientific and geopolitical data into an interactive decision engine — with dynamic chaos cascade mathematics that propagate every choice across all eight collapse vectors simultaneously.
Work IQ Intelligence Layer
NEXUS implements a Work IQ–style persistent memory system. Every decision is stored as a semantic memory entry with threat context, outcome delta, and behavioral pattern tags. This history is injected into every Claude API prompt as structured context — exactly how Microsoft Work IQ builds grounded, personalized responses from your meeting history, emails, and documents. The result: AI analysis that evolves with you across sessions.
Chaos Cascade Mathematics
Unlike static simulators, NEXUS uses a weighted cross-vector impact matrix. When you choose a climate response, it automatically recalculates probabilities across economic, pandemic, and cyber vectors — because in reality, planetary boundaries are interconnected. A geoengineering decision spikes resource conflict probability. A nuclear de-escalation reduces economic stress. Every choice ripples.
